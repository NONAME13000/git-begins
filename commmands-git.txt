➔ Cloner un repository :
    git clone <URL_du_dépôt> [<nom_du_dossier_de_destination>]

➔ Les 3 commandes de push :
    git add .
    git commit -am "Commentaire"
    git push

➔ Ajouter la commande qui permet de pull.
    git pull origin nom_de_la_branche

➔ Pour créer une nouvelle branche
    git branch nom_de_la_nouvelle_branche

➔ Pour basculer sur une autre branche
    git checkout nom_de_la_branche_cible
